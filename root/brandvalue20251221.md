# 品牌价值说明

从网名由来、预判未来、使命愿景、个性签名四个维度，进一步的解释什么叫作 “好系统” ？

## 网名由来

相比于过去的，数据库 + CURD 那种简单的、碎片化的那种小系统， <br/>
新的时代（网速 15M/s 、 20M/s 、 30M/s 、 越来越高）， <br/>
系统其实是可以更高级的，甚至一切才刚刚开始...

```text
当有一天，你用上了高维的、多彩的、富信息的、体系化的新系统，
你才能真正意识到：
以前那种：这个表、那个表、一堆表、穷信息的、碎片化的旧系统，
是多么低级、甚至是很幼稚。真的是「初步信息化」，没什么意思。

如果需要等到很多年后，才能明白这段话的意思：
那么：到时候你可能会心疼这些年的高网速、好电脑的环境...竟然...
都成了小系统的殉葬品，这些年失去的美好，再也无法挽回了，输了...

这就好比你今天才从「按钮手机」换到「智能手机」一样，当然输了...
时代已经巨变，悄无声息，很多人还没反应过来，不知不觉就过去了...

网速 20M/s 就已是「隔世巨变」了！而且是「隔了好几世的巨变」...
200K/s -> 2M/s -> 4M/s -> 10M/s -> 15M/s -> 20M/s ...
你用的甚至是适用于 2M/s 这种网速的，都是上好几辈子的系统了...
早就淘汰了、早该换代了...都算不上鸟枪换炮了，而是弓箭换战舰...

适配新时代的系统，高维、多彩、灵动，高信息密度才是真正的系统...
以前只是「初步信息化」，现在才是「信息高级化」一切才刚刚开始...
```

以后这个行业的痛点是：

网速，相当于从过去的「涓涓细流」变成了「汪洋大海」，代表着时代的美好， <br/>
一般的小系统，相当于「涓涓细流」的「小水龙头」而已，而且品质一般不高， <br/>
根本就接不下「汪洋大海」所包含的时代美好，通俗的讲：系统和时代不匹配。

觉察力高的人能够感知：

小系统根本就配不上自己的网速，太过粗陋，根本就不是这个时代该用的系统， <br/>
那都是以前 2M/s 的那种网速下， 初步信息化的小网页，现在用就是辜负时代。

***

所以：

我很有预见的、深谋远虑的，在网速还是 4M/s 的时候，一直在筹谋未来系统， <br/>
一款真正适配高网速环境的，大吞吐的、不一样的系统，以解决适配时代问题。

即：

只要你网速能达到 20M/s ， 甚至更高， 那么使用我设计的这个系统会更高级， <br/>
因为我设计的这个系统，大吞吐、富信息、多源异步，能真正用上你的高网速。

本质是：

让使用者在工作中，单位时间内获得的信息密度更高！工作品质自然就上去了！

甚至是：

使用我设计的这个系统，仅一天的数据吞吐量，一般小系统一年时间都用不到！

***

可以这样意境类比：

「网速」相当于「油」，「芯片」相当于「发动机」，「系统」相当于「车」， <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;时代赋能、时代应允：「网速/油」，不要钱的、无限的、没有上限... <br/>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;时代赋能、时代应允：「芯片/发动机」，都是飞机用的、没有上限...

那么，在这种时代下：

「系统/车」，应该是能够尽最大可能：烧油、烧CPU，那还搞什么「省油车」？

所以：在这种预见、这种洞察、这种思想，时代巨变的背景下，我做了这个事：

为认证机构设计全新的现代级信息管理系统，原则：用上高网速、用上好芯片。 <br/>
但一切很难，充满了挑战和困难、不容易，但却值得去做，所以叫作「修行」。 <br/>
我相信它将是从「禾苗」到「参天大树」的成长过程，所以前面加一个「麦」。 <br/>

所以：我的网名叫「麦修行」，就是专做这个、在这个行业里「修行」的意思。

***

## 预判未来

对于认证机构的信息系统而言，未来30年的主要矛盾是：

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; “系统**极小**、**极low**，与**极高**网速、**极好**电脑的「时代环境」不匹配的矛盾”

> 「时代环境」需要 “航母” 、去享受 “汪洋大海” ，而你的系统只是 “小木船”

现在落笔是2025年，直至2055年，至少30年，对于系统而言，认证机构拼的是：

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; **谁能用「时代之力」、谁能接「时代之美」、谁能享「时代之好」...**

> 小系统、没积累、没沉淀、糊弄做，肯定是不行的，必然接不了时代的美好

#### 输赢判断，非常简单

“我这个系统，放到2015年的网速、电脑，一样用没问题” === 100%输掉时代

“我这个系统，要不是现在的网速、电脑，可能都带不动” === 可能赢得时代

#### 为什么是可能赢得？

如果真是：“现在的高网速、电脑，才能带得动的系统”，如果你在用这种系统，

那么：大概率是赢的，因为“滥竽充数”大概率是做不到这种层次、这种境界的。

***
